# Borrador Presentacion UCM

## Estructura

- `index.html`: punto de entrada de la presentacion.
- `assets/css/styles.css`: estilos globales y variantes por tipo de slide.
- `assets/js/presentation.js`: carga modular de slides y navegacion.
- `slides/`: un archivo HTML por diapositiva.

## Ejecucion local

1. Abre una terminal en `borrador_presentacion_ucm`.
2. Ejecuta `python3 -m http.server 8000`.
3. Abre `http://localhost:8000`.

## Edicion

- Reemplaza texto Lorem Ipsum dentro de cada archivo en `slides/`.
- Sustituye contenedores `Contenedor para ...` por imagenes, graficos o capturas reales.
- Ajusta tokens visuales en `assets/css/styles.css` si necesitas afinar el estilo.
